# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
To explore and visualize the Iris dataset using data analysis and visualization techniques.

## Dataset Used
Iris Dataset from Seaborn

## Techniques Used
- Data inspection
- Descriptive statistics
- Scatter plots
- Histograms
- Box plots

## Libraries Used
- Pandas
- Seaborn
- Matplotlib

## Key Findings
- Iris-setosa is clearly separable from other species.
- Petal features provide better classification information.
- Box plots helped identify feature spread and outliers.
