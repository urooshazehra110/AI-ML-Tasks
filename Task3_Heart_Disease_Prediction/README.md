# Task 3: Heart Disease Prediction

## Objective
Predict whether a person has heart disease based on medical attributes.

## Dataset
Heart Disease UCI Dataset (Kaggle)

## Method
- Data preprocessing (missing values handling)
- Exploratory Data Analysis (EDA)
- Feature encoding
- Decision Tree Classifier

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC-AUC Score
- Classification Report

## Key Results
- Important features: exang, chest pain type, ca, oldpeak
- Model achieved good classification performance

## Tools Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
