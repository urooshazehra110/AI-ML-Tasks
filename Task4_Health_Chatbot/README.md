# Task 4: Health Query Chatbot

## Objective
Build a chatbot that answers general health related questions using a free open-source LLM.

## Model Used
Mistral-7B-Instruct (Hugging Face Transformers)

## Methodology
- Prompt engineering to control responses
- Safety filtering for harmful queries
- Conversational AI using text generation pipeline

## Features
- Friendly medical assistant behavior
- Safe response filtering
- Interactive chat system

## Tools Used
- Python
- Transformers (Hugging Face)
- PyTorch

## Key Learnings
- How LLMs respond to prompts
- Importance of safety in AI systems
- Building conversational AI systems
