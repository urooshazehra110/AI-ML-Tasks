# Task 6: House Price Prediction

## Objective
Predict house prices using property features such as area, bedrooms, bathrooms, and location.

## Dataset
Kaggle House Price Prediction Dataset

## Methodology
- Data preprocessing
- Handling missing values
- One hot encoding for categorical variables
- Linear Regression model training

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Key Results
- Model learned relationships between features and price
- Bathroom and Air conditioning features were most important

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
